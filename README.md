## Install laravel

1. Clone project:
  ```sh
     git clone https://github.com/juanpablobernardez/docker-wordpress
  ```

2. Init services:
  ```sh
    docker compose up -d 
  ```
3. Access to home page in the browser: http://localhost and follow the instructions to configure wordpress.
